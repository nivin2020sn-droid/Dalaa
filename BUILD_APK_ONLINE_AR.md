# 📱 بناء الـ APK أونلاين عبر GitHub Actions (بدون Android Studio)

هذا الدليل بالعربي لبناء **Debug APK** لتطبيق **Dalaa Beauty** مباشرة على خوادم GitHub (مجاناً)، ثم تنزيله وتثبيته على جوالك.

---

## ✅ ما ستحصل عليه

- ملف `dalaa-beauty-debug.apk` جاهز للتثبيت على أي جهاز أندرويد.
- كل مرة تعدّل على الكود وترفعه على GitHub → يُبنى APK جديد تلقائياً.
- **بدون** تنزيل Android Studio ولا Java ولا أي شيء على جهازك.

---

## 🪜 الخطوات (مرة واحدة فقط)

### 1) ارفع المشروع إلى GitHub

من داخل شات Emergent: اضغط زر **"Save to GitHub"** في شريط الإدخال.
- اختر اسم Repository (مثلاً: `dalaa-beauty`).
- اجعله **Private** أو **Public** حسب رغبتك (كلاهما مجاني للـ Actions لحد معقول).

> سيرفع المشروع كاملاً مع ملف `.github/workflows/android-debug-apk.yml` الذي أعددته لك.

### 2) شغّل البناء (Build)

بعد الرفع، افتح المستودع على GitHub:

1. اضغط على تبويب **Actions** في الأعلى.
2. لو ظهرت لك رسالة "Workflows aren't being run on this forked repository" اضغط **I understand my workflows, go ahead and enable them**.
3. من القائمة اليسرى اختر **Build Android Debug APK**.
4. اضغط زر **Run workflow** (على اليمين) → ثم **Run workflow** الأخضر.
5. انتظر ~8–12 دقيقة (أول مرة أطول لأنه يحمّل Android SDK).

### 3) نزّل الـ APK

عند انتهاء البناء بعلامة ✅ خضراء:

1. اضغط على اسم البناء (آخر Run).
2. انزل لأسفل الصفحة حتى قسم **Artifacts**.
3. اضغط على **dalaa-beauty-debug-apk** → سيتنزل ملف ZIP.
4. فك الضغط → ستجد **`dalaa-beauty-debug.apk`**.

### 4) ثبّت الـ APK على جوالك

1. انقل الملف إلى جوالك (واتساب إلى نفسك، Google Drive، كابل USB، …).
2. من الجوال: **Settings → Security → Install unknown apps** → فعّل الإذن للمتصفح أو مدير الملفات.
3. افتح ملف الـ APK من الجوال → **Install** → **Open**.
4. ادخل بحساب المسؤول الافتراضي:
   - **Email:** `admin@salon.com`
   - **Password:** `admin123`
5. **غيّر كلمة المرور فوراً** من داخل التطبيق بعد أول دخول.

---

## 🔁 عند تحديث الكود لاحقاً

أي تعديل على الكود → اعمل **Save to GitHub** مرة أخرى → سيُبنى APK جديد تلقائياً (لا تحتاج تضغط Run يدوياً، لأنه مضبوط على `push`).

ادخل تبويب **Actions** → آخر Run → نزّل الـ APK الجديد.

## ⚠️ ملاحظات مهمة

| الموضوع | التفاصيل |
|---------|----------|
| **Debug APK** | موقَّع بمفتاح تطوير، لا يصلح للنشر على Google Play، لكن يعمل 100% على جوالك الخاص. |
| **المدة الحرة في GitHub Actions** | 2000 دقيقة/شهر للمستودعات الخاصة، لا محدودة للمستودعات العامة. بناء واحد يأخذ ~10 دقائق. |
| **حجم الـ APK** | ~6–10 ميجابايت. |
| **البيانات** | محلية 100% في جوالك (IndexedDB). لا تُرفع على أي خادم. |
| **النسخ الاحتياطي** | استخدم زر **Backup / Restore** داخل التطبيق (صفحة الإعدادات). |

## 🐞 إذا فشل البناء

افتح البناء الفاشل على GitHub → اضغط على خطوة مكتوب بجنبها ❌ → انسخ الخطأ وارسله لي هنا وسأصلحه.

الأخطاء الشائعة وحلولها:

| الخطأ | الحل |
|-------|------|
| `Gradle failed: SDK not found` | نادر، أعد تشغيل الـ Workflow. |
| `yarn install failed` | تأكد أن ملف `frontend/yarn.lock` مرفوع على GitHub. |
| `cap add android failed` | احذف مجلد `frontend/android/` من المستودع لو كان موجوداً جزئياً. |
