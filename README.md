# 💅 Salon Mobile App — Offline Android APK

Beauty salon accounting app packaged as a **fully offline Android APK**. All data stays on the device — no server, no internet required.

**Stack:** React · Tailwind · Shadcn UI · Dexie (IndexedDB) · bcryptjs · Capacitor (Android)

---

## 📁 Layout

```
.
├── frontend/                  ← React app + Capacitor
│   ├── src/
│   │   ├── db/                ← IndexedDB schema + seed
│   │   ├── services/          ← local business logic (replaces backend)
│   │   ├── pages/             ← UI pages
│   │   └── api.js             ← in-app router (no HTTP)
│   ├── capacitor.config.json  ← Android app config
│   ├── build/                 ← generated by `yarn build`
│   └── android/               ← generated by `npx cap add android` (run on your PC)
├── package.json               ← root scripts
├── MOBILE_BUILD.md            ← full step-by-step APK build guide
└── README.md
```

---

## 🚀 Quick Build to APK

```bash
yarn install:all
yarn build
yarn cap:add-android   # first time only
yarn cap:sync
yarn cap:open          # opens Android Studio → Build → Build APK
```

👉 **Full instructions with troubleshooting → [`MOBILE_BUILD.md`](./MOBILE_BUILD.md)**

## 💾 Data Storage

All data lives in **IndexedDB** inside the Android WebView. It persists across:
- App restarts ✅
- Device reboots ✅
- App updates ✅

It is wiped only if the user **uninstalls** the app or **clears app data** from Android Settings.
